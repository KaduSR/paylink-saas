# PayLink SaaS 💳

> Plataforma de assinatura recorrente com checkout Stripe e dashboard financeiro.

## Stack
Next.js • Stripe • Webhooks • Docker • PostgreSQL • Redis

## Funcionalidades
- Checkout Stripe integrado
- Webhooks para eventos de pagamento
- Dashboard financeiro com graficos
- Notas fiscais automaticas
- Filas de processamento assincrono
- Multi-tenancy

## Metrica
Sincronia de pagamentos via webhook em menos de 1 segundo.
